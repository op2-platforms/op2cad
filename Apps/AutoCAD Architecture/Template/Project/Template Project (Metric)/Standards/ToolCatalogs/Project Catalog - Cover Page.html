The AutoCAD® Architecture Project Tool Catalog may contain tools that refer to architectural object styles and AEC design content.  You can use this catalog as one place to come for all tools specific to a project.
 
CAD Managers - By serving up content in the form of tools, you can keep your average users out of Style Manager and Design Center where editing of the raw styles can occur.
 
To create your own architectural object tools simply drop the style from Style Manager, or drop the actual object from a saved/central drawing file.  To create your own AEC design content tools, just drag/drop from the 'AEC Content' tab in DesignCenter onto a tool palette in your workspace.
